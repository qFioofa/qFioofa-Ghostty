# qFioofa-Ghostty

Personal Ghostty terminal config.

# Why Ghostty

- Fast terminal immulater written in `zig`
- Build in fonts
- Custom shader support
    - Cursor animation

# Install

With `git clone`

```bash
git clone git@github.com:qFioofa/qFioofa-Ghostty.git
cd ./qFioofa-Ghostty
bash deploy_config -r
```

# Clean

Delete config

```bash
rm -rf ~/.config/ghostty/
```
